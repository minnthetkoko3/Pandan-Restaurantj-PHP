A dynamic restaurant website built to demonstrate backend development skills using the LAMP stack approach. This project features a fully functional backend that integrates with a relational database to manage dynamic content.

Tech Stack:

Backend: PHP

Database: MySQL

Frontend: HTML/CSS (and JavaScript if applicable)

Key Features:

Dynamic menu rendering from the database.

CRUD (Create, Read, Update, Delete) operations for managing restaurant data.

Secure database connectivity using MySQL.
